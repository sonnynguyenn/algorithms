# Vector in linear algebra
In linear algebra, a vector can store information about a point (position) or a direction. Overall, whatever information is stored inside a vector, the number always could be understood as an offset from origin. <br>
Let's say, we are working in 2D of linear algebra in which we have two axes, Y and X.
